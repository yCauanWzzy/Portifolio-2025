Organizador Automático de Pastas

Descrição:
Este projeto foi desenvolvido em Python com o objetivo de automatizar a organização de arquivos em uma pasta. O aplicativo escaneia um diretório selecionado e move os arquivos para subpastas específicas, de acordo com seu tipo (como imagens, vídeos, documentos, executáveis, entre outros).

Funcionalidades:

  • Classificação automática por extensão de arquivo (.jpg, .pdf, .mp4, etc)
  • Criação de subpastas automaticamente
  • Interface de linha de comando simples e funcional
  • Código leve e eficiente, ideal para uso pessoal ou corporativo

Tecnologias utilizadas:
Python 3, módulos os e shutil, e PyInstaller para gerar o .exe

from utils.file_utils import organizar_pasta from pathlib import Path import json def carregar_config(): config_path = Path("config.json") if config_path.exists(): with open(config_path, "r", encoding="utf-8") as f: return json.load(f) else: return { "pasta_destino": "", "usar_data": False } def salvar_config(config): with open("config.json", "w", encoding="utf-8") as f: json.dump(config, f, indent=4) def main(): config = carregar_config() print("=== Organizador de Arquivos ===") caminho = input("Digite o caminho da pasta (ou deixe em branco para usar anterior): ").strip() if caminho: config["pasta_destino"] = caminho escolha_data = input("Organizar também por ano de criação? (s/n): ").lower().startswith("s") config["usar_data"] = escolha_data salvar_config(config) organizar_pasta(Path(config["pasta_destino"]), usar_data=escolha_data) if __name__ == "__main__": main()

Motivação:
Este projeto surgiu da necessidade de manter pastas como "Downloads" ou "Área de Trabalho" organizadas sem esforço manual. Ideal para quem lida com muitos arquivos e deseja mais produtividade no dia a dia.
